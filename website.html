HTML DAN CSS

1. HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan isi halaman web. HTML menentukan apa saja yang ada di dalam sebuah halaman, seperti:

  • Judul
  • Paragraf
  • Gambar
  • Tautan (link)
  • Tabel, daftar, dan form

Contoh HTML

2. CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web yang dibuat dengan HTML. CSS menentukan bagaimana tampilannya, seperti:

  • Warna text dan background
  • Ukuran dan jenis font
  • Tata letak
  • Animasi
  • Jarak

Contoh CSS

Hubungan HTML dan CSS

HTML dan CSS memiliki hubungan yang saling melengkapi dalam pembuatan halaman web. HTML berfungsi untuk menyusun struktur dan isi halaman web, seperti judul, paragraf, gambar, dan daftar. Sementara itu, CSS digunakan untuk mengatur tampilan dari elemen-elemen HTML tersebut, seperti warna, ukuran, tata letak, dan jarak. Tanpa HTML, halaman web tidak memiliki struktur, dan tanpa CSS, tampilan halaman web menjadi sederhana dan kurang menarik. Oleh karena itu, HTML dan CSS digunakan bersama agar halaman web dapat berfungsi dengan baik dan terlihat rapi serta menarik.
